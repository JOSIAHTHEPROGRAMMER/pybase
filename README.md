# PyBase

PyBase is a lightweight relational database engine built in Python for educational and systems-level learning purposes.

## Goals

- File-based persistence
- SQL-like query parsing
- Custom storage engine
- B-Tree indexing (planned)

This project is built from scratch to explore how relational databases work internally.# PyBase
